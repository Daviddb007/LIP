GET /api/v1/estadisticas

Estadísticas completas del ecosistema: participaciones, departamentos, sectores, SRIE, tendencias.

Público · Sin autenticación

{
    "total_participaciones": 10,
    "total_departamentos": 3,
    "total_politicas": 6,
    "sectores": [...],
    "problemas": [...],
    "srie_pilares": [...],
    "srie_urgencia": [...],
    "srie_impacto": [...],
    "tendencia": [...]
}
GET /api/v1/participaciones

Listado paginado de participaciones ciudadanas. Requiere autenticación.

Requiere token
Parámetros: ?page=1&per_page=20§or=Educación&departamento=Antioquia 

{
    "data": [{ "id": 1, "departamento": "...", "propuesta": "...", "srie_pilar": "..." }],
    "total": 100,
    "page": 1,
    "pages": 5
}
GET /api/v1/politicas

Listado de políticas públicas activas. Filtrable por sector.

Público · ?sector=1

GET /api/v1/politicas/{id}

Detalle completo de una política pública por ID.

Público

GET /api/v1/sectores

Todos los sectores activos del ecosistema.

Público

GET /api/v1/armonizacion

Matriz de armonización estratégica: comparación participación vs políticas vs ODS.

Público

GET /api/v1/analitica

Analítica avanzada: nube de palabras, clusters, tendencias, predicciones.

Público

POST /api/v1/clasificar

Clasifica un texto usando el motor SRIE. Recibe JSON con los datos de participación.

Público · Rate limit: 10/min

// Request
{ "problema_principal": "...", "propuesta": "...", "sectores": [1,2] }

// Response
{
    "pilar": { "nombre": "Educación", "color": "#8b5cf6", "confianza": 80 },
    "urgencia": { "nivel": "Alta", "color": "#ea580c" },
    "impacto": { "nivel": "Regional", "color": "#2563eb" },
    "explicacion": "..."
}
POST /api/v1/token

Genera un token de acceso. Requiere token de administración en el header.

Requiere admin token

// Request
Header: Authorization: Bearer <admin_token>
Body: { "nombre": "Mi App", "role": "lectura" }

// Response
{
    "id": 1,
    "nombre": "Mi App",
    "token": "a1b2c3...",
    "role": "lectura",
    "activo": true
}
Ejemplo de uso

Aquí tienes ejemplos en diferentes lenguajes:

import requests

# Sin autenticación
resp = requests.get("https://inteligenciapublica.stonelytics.tech/api/v1/estadisticas")
print(resp.json())

# Con autenticación
headers = {"Authorization": "Bearer TU_TOKEN"}
resp = requests.get("https://inteligenciapublica.stonelytics.tech/api/v1/participaciones", headers=headers)
print(resp.json())
# Estadísticas (público)
curl https://inteligenciapublica.stonelytics.tech/api/v1/estadisticas

# Participaciones (con token)
curl -H "Authorization: Bearer TU_TOKEN" \
  https://inteligenciapublica.stonelytics.tech/api/v1/participaciones

# Clasificar con SRIE
curl -X POST -H "Content-Type: application/json" \
  -d '{"propuesta":"Mejorar educación"}' \
  https://inteligenciapublica.stonelytics.tech/api/v1/clasificar
// Sin autenticación
fetch('https://inteligenciapublica.stonelytics.tech/api/v1/estadisticas')
  .then(r => r.json())
  .then(console.log)

// Con autenticación
fetch('https://inteligenciapublica.stonelytics.tech/api/v1/participaciones', {
  headers: { 'Authorization': 'Bearer TU_TOKEN' }
}).then(r => r.json()).then(console.log)
